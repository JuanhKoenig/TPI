ANTES DE EMPEZAR



es importante que se abra el programa con Visual Studio Code y paises.csv en la misma carpeta o

el programa va a iniciar con un error o aveces no inicia.





Este es un programa que edita un archivo CSV de paises para agregar, editar, buscar,  y ordenar datos

estas son las siguientes funciones:





1. ver datos

&#x09;para ver el los elementos del archivo.csv y filtrar datos



2\. Agregar país

&#x09;pide datos de nombre, población, superficie, y continente para agregar al archivo



3\. Actualizar datos

&#x09;editar datos de algún país



4\. Buscar

&#x09;buscar algún país, y si esta en la lista, muestra sus datos



5\. Estadísticas

&#x09;muestra el resumen de las estadísticas con respecto a mayor y menor población

&#x09;y superficie, países por continentes, y promedios

6\. Ordenar

&#x09;Ordena los datos del CSV. Los archivos quedan guardados en el ultimo orden que

&#x09;se ordenó



7\. Salir

&#x09;Guarda los datos y cierra el programa
